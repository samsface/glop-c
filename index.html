About

Glop-c is an open source C engine for simulating deformable soft bodies in 2D. It's developed by Sam Murray under the MIT License (MIT). While this license does not require acknowledgement, I would appreciate a credit to Glop-c in your product.

Glop-c Features

Physics

  • Elastic & destructable constraints
  • SAT collision solving
  • Fast broad-phase using AABB with spatial buckets
  • Water & air resistance
  • Thrusters, balloons, oscillators and other special body types

System

  • OSX & Unix platforms
  • Multi-core support
  • Light-weight code base at less than 100kb

Testbed

  • OpenGL GUI on freeglut
  • Scriptable dev console
  • JSON serialization

Documentation

  • User Manual
  • Doxygen full brief

Installation

Run the following code:

git clone https://github.com/samsface/glop-c.git

and copy the header and c files to you source code tree.

Alternatively, run make test to compile and run the test environment.
